# Hello Craftwave · Claude Code 插件

Hello Craftwave —— 由匠播 Craftwave 一键发布的Skill。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/hello-craftwave-claude-plugin
/plugin install hello-craftwave@hello-craftwave-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Hello Craftwave**。本仓库由
[匠播 Craftwave](https://craftwave.app) 按 Claude Code 插件市场规范自动生成。
